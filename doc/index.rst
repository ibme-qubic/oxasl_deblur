OXASL_DEBLUR - Tool for deblurring of ASL-MRI data
==================================================

OXASL_DEBLUR is a package for performing deblurring on ASL-MRI data

To install, if you have an FSL installation::

   fslpython -m pip install oxasl_deblur --user

If you do not have an FSL installation, you can install into any Python environment using::

   pip install oxasl_deblur
   
Be aware that some functionality requires FSL to be installed.
